## Functionality:

0. Sign up / log in (JWT-token)
1. Add / remove / rename todolist
2. Find todolists
3. Add / remove / edit / complete task
4. Filter tasks
5. Find tasks
6. Editing task includes title, description, images, date change
7. Black / white theme
